CPU Simulator

CPU Simulator does exactly what the name implies: simulates a computer processor. The program takes an input of a series of numbers (given as a csv), which it translates into binary. These binary numbers are all parsed as instructions by an ISA (instruction set architecture), then executed by the program. Through the use of the preprocessor, it's possible to change how many instructions are fetched, executed, and removed from the queue each 'clock cycle.''

Vending Machine

Vending Machine is a program intended to run on a physical vending machine to provide all the functions necessary, including setting prices, purchasing products, and even a backdoor to receive products for free :). Without a physical vending machine, the input must be set prior to runtime and the output is in a form of a digital signal graph, but of course, when ran on a physical machine, inputs would be relayed from the PIN pad, and outputs would trigger mechanisms to release a product.

Star Trek Episodes

At the moment, it is left as a proof of concept (with TOS and TAS done, and TNG partially done) due to the large amount of Star Trek episodes.
